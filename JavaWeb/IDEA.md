# 快捷键

Ctrl + R : 替换  

Ctrl + Shift + R 按照不同的目录进行替换



Alt + Shift + 上(下) : 将代码上(下)移   

Ctrl+Alt+V 自动生成变量名



Ctrl + p 查看参数
